【小王子】－〔第十二章〕　　　　　
　

　　下一顆行星住了一位酒鬼。這次的拜訪很短，可是卻使小王子非常憂鬱。　　　
　　「你在那裡幹什麼？」他問酒鬼。
他看見那位酒鬼默默地坐在一大堆酒瓶和一大堆空瓶 前面。　　　

　　　　　　
　　「我在喝酒。」酒鬼滿臉愁容地回答說。　　　
　　「你為什麼喝酒？」小王子問。　　　
　　「為了要忘掉。」酒鬼回答。　　　
　　「忘掉什麼？」小王子追問他，一邊可憐他。　　　
　　「忘掉我的可恥。」酒鬼坦白的說，同時低下了頭。　　　
　　「可恥什麼？」小王子問他，一心想救他。　　　
　　「喝酒可恥！」酒鬼說完他的話後，一言不發，像掉進沈默的深淵。　　　
　　小王子不知所措地走了。　　　
　　旅途上他不斷的對自己說：「大人們實在非常非常奇怪。」
