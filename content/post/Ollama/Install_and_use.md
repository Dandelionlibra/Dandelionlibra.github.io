# 介紹 Ollama
Ollama 是一個能在 本地（Windows/Mac/Linux）執行大型語言模型（LLM）和 Vision Language Model（VLM） 的框架。

* 開源工具
* 在本地端運行大型語言模型
* 離線特性以保護隱私
